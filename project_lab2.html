색상 원리와 풀-컬러 Visual-MIMO 기술 융합 연구

참여 기간 2017.09 ~ 2019.08

색상 원리와 풀-컬러 Visual-MIMO 기술 융합 연구

Visual-MIMO(Multi Input Multi Output) : 다양한 데이터를 LED를 변화시켜 송신, 사람의 눈은 일정 속도 이상으로 빠르게 변하는 빛을 단색으로 인지하지만, 카메라를 이용하게 되면 변하는 데이터를 읽을 수 있다. 이를 이용하여 데이터를 송수신하려는 연구이다.

Visual-MIMO Block Diagram

제작 작품 : 심박 및 산소포화도 측정 Visual-MIMO Test Module

■ 제작 기간 : 2018.03 - 2018.10, 약 8개월

■ 구성: LED Array(LEA) + Arduino + PPG Sensor + Android Application

제작한 Visual-MIMO Block Diagram

■ 개발 인원 : 2명

  • 기기 제작 및 색상 분석 알고리즘 : 1명
  • 어플리케이션 및 서버 제작 : 1명
▶ 맡은 역할 : 어플리케이션 및 서버 제작
안드로이드 기기의 카메라를 조작하여 이미지를 서버로 보낼 수 있는 안드로이드 어플리케이션을 제작하였다.
TCP/IP 서버를 제작하여 어플리케이션의 이미지를 서버에 저장, 분석된 데이터를 어플리케이션으로 송신 하도록 하였다.
